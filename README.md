## 👋 Обо мне

Frontend-разработчик, четвёртый год в вебе. Собираю SPA на **React + TypeScript** и **Vue 3 / Nuxt 3** — от аккуратной адаптивной вёрстки до модульной архитектуры и интеграции с API. Люблю чистый, читаемый код и интерфейсы, в которых всё на своих местах.

- 🎯 В фокусе: модульная архитектура (FSD), работа с REST API, state management, продуманный UI/UX
- ⚡ Использую AI-assisted workflow, чтобы доставлять быстрее и качественнее
- 🌱 Прокачиваю TypeScript, анимации и продуктовое мышление
